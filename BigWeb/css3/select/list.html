CSS选择器
伪类与伪元素
  • 1、规范中伪类一个冒号表示，伪元素两个冒号，为了兼容有些伪元素也能用单冒号
  • 2、常见伪类选择器如 ：hover、:first-child、：nth-child(2n)等
  • 3、常见伪元素选择器如 E::selection、E::after、E::first-letter等
属性选择器
  • 1、与正则类似、用中括号
  • 2、E[attr*="val"] 选择具有att属性且属性值为包含val的字符串的E元素
关系选择符
  • 1、包含E F | 子 E>F
  • 2、E紧贴的下一个F E+F | E后面所有同级F元素 E~F
元素选择符
  • 1、 * div #xxx .xxx 等
属性
颜色表示方法
  • p{color:green;}
    p{color:#ff0000;}
    p{color:rgb(0,0,0);}
    下面的IE8一下不支持
    p{color:rgba(0,0,0,.5);}
    p{color:hsl(240,50%,50%);} 色调、饱和度、亮度
    p{color:hsla(240,50%,50%,.5);}
    p{color:transparent;} 透明色
text-shadow文本阴影
  • 1、text-shadow:none| px{2,3} && color,px{2,3} && color2
  • 2、参数:左右偏移,上下偏移,模糊程度,颜色
text-overflow文本溢出省略号
  • /*单行*/
    overflow: hidden;
    text-overflow: ellipsis;
    white-space:nowrap;
  • /*多行溢出省略号显示 不支持火狐*/
    overflow: hidden;
    text-overflow: ellipsis;
    display: -webkit-box;
    -webkit-line-clamp: 1;
    -webkit-box-orient: vertical;
border-radius圆角边框
  • 四参数:上左 - 上右 -下右 - 下左
  • 三参数:上左 - 上右,下左 - 下右
  • 双参数:上左,下右 - 上右,下左
  • 一参数:四个角
  • border-radius:5px/10px 水平5像素，垂直10像素
  • border-top-right-radius:5px/10px 上右角水平5像素，垂直10像素
box-shadow边框阴影
  • 1、box-shadow:none|inset? [ px{2,4} && color ], [ px{2,4} && color ]
  • 2、px参数:水平偏移,垂直偏移,模糊程度,向外延申
border-image
  • 1、主要三个参数:路径(source)、切割(slice)、平铺(repeat)
  • 2、切割的fill决定盒子背景要不要用图片除边框宽度的中间部分填充
  • 3、border-image-repeat : [ stretch | repeat | round ]{1水平,2垂直}
  • 4、border-image: url(/border-image.png) 27 fill repeat repeat;
